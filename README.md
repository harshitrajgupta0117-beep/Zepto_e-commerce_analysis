# Zepto_e-comZepto Product & Inventory Analysis (SQL)
Project Overview
This project performs a deep-dive analysis of an e-commerce dataset inspired by Zepto, a leading quick-commerce platform.
The goal is to evaluate pricing strategies, inventory health, and category performance to provide actionable business insights.

Technical Skills Demonstrated
Data Aggregation: Calculating total revenue and category-wise statistics.

Conditional Logic: Using CASE statements to segment products into price tiers.

Data Filtering: Identifying stock-out risks and specific category trends.

Performance Metrics: Calculating price-per-gram to understand value-based pricing.

Key Business Questions Answered
Revenue Potential: What is the total market value of the current inventory?

Inventory Risk: Which categories are suffering most from stock-outs?

Pricing Strategy: How are products distributed across Budget, Mid-Range, and Premium tiers?

Promotional Analysis: Which products carry the highest discounts and in what categories?
